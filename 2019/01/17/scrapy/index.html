Scrapy:
Pip install scrapy
Scrapy 第三方爬虫框架 市面上 用的最多的
内部集成了 TWISTED异步网络框架 —> 实现的多线程爬虫（异步并发）
设置了五大模块:
spider（爬虫类） —>1、start_urls 初始入口 2、解析数据
Engine(引擎） —>调度其他四个模块
Scheduler(调度器) —>1、request 排队 入队列 2、request对象出队列
Downloader（下载器）—>下载数据
pipeline（管道） —>存储数据
工作原理：
Engine—->>:engine
1、入口 spider –start_urls(request)–>engine–>调度器
2、调度器–>request–>入队列
3、调度器–>request–>出队列–>engine
4、engine–>request–>下载器
5、下载器–>response –>engine
6、engine –>response –>spider（解析数据）—->
1、new_url多个就返回第一个条
2、data（有数据）—>engine—>pipline(存储数据)
scrapy代码操作：
1、创建爬虫项目—->scrapy startproject 项目名称
2、建爬虫 —>1、cd 项目路径 2、scrapy genspider 爬虫名字 爬虫域名
3、运行爬虫项目 —>1、a：cd spiders b: scrapy runspider 爬虫文件。Py
2、scrapy crawl 爬虫名字
4、解析数据 —>item1、 filed写错了会报错
2、response.xpath().extract()
5、数据存储—>1、return item（在爬虫数据给了引擎 引擎给了管道） —>spider-item-engine-pipeline
2、jsonitemExpoter（导出器）
3、file文件写入 –》wb（二进制写入）
4、exporter(file) （导出）
5、start_exporting （开启）
6、export_item （导出）
7、finish_exporting() （关闭）
8、settings开启管道 （默认不用管道）
