发布于 2019-09-12   更新于 2019-09-12 Git 热度 0 喜欢 0

建议大家在学习前期最好不要用《Sourcrtree》——Git管理工具

为啥说最好前期不要用呢，是因为大家本来对git不熟悉的话，还是建议多敲敲命令，多熟悉一下git，对项目的结构会认识很多，很深。（只是个人意见）如果想用的话，https://liyukuo.github.io/2019/09/12/git版本控制图形化界面推荐/

 这种工具个人感觉是给那些老程序员用的，对于新手虽然说很友好，但是不懂git原理的话，用起来提心吊胆， 并且如果一不小心点错了。你也不知道怎么改，老手直接一行命令解决照样用工具，但是你不行，所以建议大家还是先熟悉了git之后，在使用工具，我周围的都是命令，只有我一个人使用工具，（但是我觉的贼好使[/偷笑]）

这个工具支持Windows和Mac,官方有下载地址，大家可以自行去找。Sourcrtree官网  祝大家一切顺利！
